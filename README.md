# Click Game

A simple reaction game where players click a moving target to score points.

## Features
- Timer-based gameplay (60 seconds)
- Live leaderboard
- Google Sheets backend
- Score saving via API

## Tech stack
- HTML, CSS, JavaScript
- Google Apps Script (backend)
- Google Sheets (database)

## How it works
Scores are sent to a Google Apps Script API which stores them in a Google Sheet and returns a leaderboard.
